# Expense Tracker Application


## Setup

1. Clone the project.

```
git clone https://github.com/verai17/piggyoink.git
```
2. Install the dependencies.

```
cd piggyoink
npm install
```
 
---

## Run

1. Run project.

``` 
npm start
```

2. Visit Application.

```
http://localhost:3000/
```
